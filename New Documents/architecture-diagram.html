AUTH
NextAuth.js
Web Application — Authentication
Handles user authentication for the web dashboard. Two-layer security model: Supabase Auth manages session authentication at the UI layer, while HMAC-SHA256 signatures secure each API request to the FastAPI backend at the API layer.
SECURITY
HTTPS Webhooks — HMAC Signed
Web Application — Transport Security
All communication between the Next.js dashboard and the FastAPI backend is via HTTPS REST endpoints. Each request includes an HMAC-SHA256 signature computed from the request body and a shared secret. The FastAPI backend verifies the signature before processing any request.
API
REST API Layer
Web Application — API Interface
33+ REST endpoints exposed by FastAPI, consumed by the dashboard. Covers all worker bee operations: content generation, database queries, analytics, and orchestration. Full request/response contracts documented in api-contract.md.
