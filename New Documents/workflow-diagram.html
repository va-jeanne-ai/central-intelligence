Central Intelligence — CEO

Receives user requests and delegates to the appropriate Director via Sub-Workflow Tools. Queries the Shared Intelligence Pool for cross-department context. Aggregates Director responses into final output.

Level 3 — Directors

Each Director manages a department. They receive tasks from Central Intelligence and coordinate their Specialists via Sub-Workflow Tools. Directors aggregate specialist outputs and return structured results.

Level 2 — Specialists

Domain experts that execute specific tasks. They read from and write to the Shared Intelligence Pool using Data Table Tools. Each Specialist has deep expertise in one area.

Level 1 — Operators (Floaters)

Single-purpose data feeders that run independently. They write to shared tables (stats, transcripts, comments) and are not managed by any Director. Triggered by schedules or events.

Shared Intelligence Pool

Central data layer with 9 tables: goals, wins, pain_points, objections, content_ideas, icp, offers, comments, reference. All Specialists and Operators read/write here, creating a shared knowledge base across the entire Central Intelligence.
