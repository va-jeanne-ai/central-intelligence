1. Project Purpose

Central Intelligence is a unified AI automation platform that orchestrates business intelligence across three departments (Marketing, Sales, Fulfillment). Built on a three-level agent hierarchy, the system enables cross-domain insights, automated data flows, and strategic decision-making through a single interface.

The platform replaces isolated single-purpose AI tools with integrated specialist agents that share a common intelligence pool, enabling marketing to leverage sales insights, fulfillment wins to inform content strategy, and leadership to make data-driven priorities based on unified visibility.

Tier 1 (Central Intelligence): CEO-level orchestrator that responds to cross-department strategic questions and synthesizes insights from all three departments.

Tier 2 (Directors): Three department heads coordinate their specialists, aggregate department-level metrics, and report to Central Intelligence.

Tier 3 (Specialists + Operators): 15+ domain-expert specialists handle specific work (social media, email, leads, members, etc.) plus shared operators for transcription, generation, and updates. All read/write to 16 shared intelligence tables.

5. Key Concepts

Shared Intelligence Pool

All specialists read from and write to a common set of 16 Supabase tables. This enables:

  • Marketing specialist uses sales pain points to inform content calendar
  • Fulfillment specialist exports member wins for marketing content ideas
  • Central Intelligence synthesizes cross-department metrics for executive dashboards
  • No manual data flows between departments

Agent Hierarchy with Claude SDK

All agents (Central Intelligence, Directors, Specialists) use Anthropic Claude for reasoning. The hierarchy enables:

  • Central Intelligence: Multi-turn conversations with strategic context from all departments
  • Directors: Department-level decision making, aggregating specialist insights
  • Specialists: Domain-expert tasks (email writing, lead analysis, member tracking)

Two-Component Architecture

  • Component 1: Python agentic backend (FastAPI + Claude SDK + Celery)
  • Component 2: Next.js frontend with streaming UI and real-time dashboards
  • Shared Data Layer: Supabase PostgreSQL with RLS

Voice of Customer (VOC) Intelligence

Central Intelligence subsystem (integrated Python agents) transforms call transcripts (sales, coaching, discovery, accountability) into structured VOC data that powers marketing content, email writing, and strategic priorities.
