Central Intelligence · Agent Workflows & Sprint Plan

Last updated: 2026-04-29 8 Sprints 17 Weeks 701 Story Points

Workflow Overview

Central Intelligence is a 4-level agentic architecture that orchestrates business operations across Marketing, Sales, and Fulfillment departments. All agents (Directors, Specialists, Operators) collaborate through a Shared Intelligence Pool — a central data repository containing goals, wins, pain points, objections, content ideas, ICPs, offers, comments, and reference data.

Central Intelligence (CEO)

Top-level AI orchestrator that receives user requests, delegates to appropriate Directors, and synthesizes cross-department responses.

  • Routes tasks to Marketing, Sales, Fulfillment Directors
  • Queries Shared Pool for business context
  • Provides executive-level insights

Shared Intelligence Pool

Central data layer with 9 shared tables. All Specialists and Operators read/write here, creating unified business knowledge.

  • goals, wins, pain_points, objections
  • content_ideas, icp, offers, comments, reference

Agent Hierarchy & Workforce

The system operates through a 3-tier hierarchy: Directors manage departments, Specialists execute domain tasks, and Operators feed shared data.

Level Role Count Key Responsibilities
CEO Central Intelligence 1 Cross-department orchestration, strategic decisions, multi-query reasoning
Level 3 Directors 3 Marketing, Sales, Fulfillment — department-level reasoning & specialist coordination
Level 2 Specialists 13 Domain experts (email, social, leads, coaching, etc.) with deep knowledge
Level 1 Operators 6 Shared utilities (transcriber, ICP gen, stats updaters) serving all departments
Shared Intelligence Pool 9 tables goals, wins, pain_points, objections, content_ideas, icp, offers, comments, reference

Core Operators & Cross-Cutting Systems

Shared operators that power all departments and specialists.

CI-CORE-01: Call Transcriber

Converts video/audio into transcripts via OpenAI Whisper. Routes transcripts to appropriate analyzers (Sales, Coaching, Accountability). Shared across all departments.

CI-OPS-ICP: ICP Generator

Synthesizes Ideal Client Profiles from pain points, wins, and objections. Produces structured demographics, psychographics, and trigger data. Feeds Marketing offer/DM specialists.

CI-OPS-OFFER: Offer Generator

Creates product/service offer structures from ICP, pain points, and objections. Produces pricing tiers, positioning, and bonus recommendations for Marketing specialists.

Stats Updater Operators

Scheduled Celery tasks pulling live metrics from source systems (Airtable, Google Sheets, etc.). Includes Email Stats, Social Stats, Funnel Stats, Ads Stats, Comments Collector.

Error Handler

Centralized error logging agent. Captures unhandled exceptions across all workers, logs to bee_error_log table, alerts on critical failures.

Workflow Inventory & Coordination

How workflows interact and coordinate through the Shared Intelligence Pool and Central Intelligence orchestration.

Transcript Pipeline

  • User uploads video → CI-CORE-01 Transcriber processes → OpenAI Whisper generates transcript
  • Transcript routed by call_type (sales, coaching, accountability) to appropriate Specialist analyzers
  • Analyzers extract pain_points, wins, goals, content_ideas → written to Shared Pool
  • All Specialists read from Shared Pool for cross-domain context

Intelligence Synthesis

  • CI-OPS-ICP reads pain_points + wins + objections → generates ICP profiles
  • CI-OPS-OFFER reads ICP + pain_points → generates offer structures
  • Marketing Specialists (Social, Email, Ads, DM, Offers) read from Shared Pool to inform content generation
  • Central Intelligence queries Directors for department-level insights

Content Generation Flow

  • Marketing Director receives user request → routes to appropriate Specialist
  • Specialist reads from Shared Pool (pain_points, wins, content_ideas, ICP, objections)
  • Specialist uses Claude to synthesize content (email, social script, ad copy, offer, DM template)
  • Content returned to user or stored for review

Cross-Domain Integration

  • Sales Calls Analyzer → extracts pain_points, content_ideas to Shared Pool
  • Marketing uses these insights to create campaigns addressing sales pain points
  • Coaching Analyzer → extracts wins, pain_points to Shared Pool
  • Marketing uses wins for social proof / success stories
  • Fulfillment uses pain_points to improve member experience
